Все работы - одностраничные макеты сайтов.

Краткое описание

  • Макет выбран из бесплатного сервиса
  • В качестве картинок персонала - котики из CatApi

Краткое описание

  • Пиксельная точность для Chrome latest
  • Верстка responsive (для моб девайсов)
  • Подключены нестандартные шрифты
  • Использован препроцессор SASS
  • Использован сборщик Gulp
  • Слайдеры макета сделаны без JS (только CSS)
  • В нижней части макета есть блок картинок с заголовком "Discover holiday activity ideas".Они расположены с помощью плагина Masonry
  • Картинки взяты с помощью ajax-запроса с сервиса Pixabay
  • После блока картинок есть поле поиска. При введении в поле текста и нажатии кнопки "Search partners" отправляется запрос на получение картинок с сервиса, и картинки заменяются на новые, соответствующие нашему запросу. В качестве текста для каждой картинки используется поле word из ответа сервера
